# GymLog 🏋️

A simple, elegant workout planner and tracker for the gym. Built as a single-page web app with vanilla HTML/CSS/JS and localStorage for data persistence.

## Features

- **Create Workout Plans**: Design your own workout routines with exercises, sets, reps, and weights.
- **Track Live Sessions**: Start a session from any plan, track completed sets in real-time.
- **Workout History**: View all your completed sessions with stats and notes.
- **Progress Stats**: See your total sessions, weekly activity, and day streak.
- **Mobile-Friendly**: Responsive design that works on phones and tablets.
- **No Backend Needed**: All data stored locally in your browser (localStorage).

## Usage

1. **Create a Plan**: Go to "Plans" tab → "New Plan" → Add exercises with sets/reps/weight.
2. **Start a Session**: From Home or Plans, click "Start" on a plan.
3. **Track Your Workout**: Mark sets as completed, adjust reps/weight as needed.
4. **Save Session**: When done, complete the session and add optional notes.
5. **Review History**: See all your past workouts in the History tab.

## Tech Stack

- **HTML5/CSS3/ES6** – No frameworks, no build step.
- **localStorage** – Data persists in your browser.
- **GitHub Pages** – Free hosting.
- **Font Awesome** – Icons.
